## The language of corporate planning

To get the most out of Cord's master budgeting engine, it's crucial that you and your team speak the same financial language. Below, we explain the key accounting terms in a technical reference table:

<div style="border: 1px solid #e2e8f0; border-radius: 16px; overflow: hidden; background: #ffffff; margin-top: 32px; box-shadow: 0 4px 24px rgba(0,0,0,0.02);">
  
  <div style="display: flex; flex-wrap: wrap; border-bottom: 1px solid #e2e8f0;">
    <div style="flex: 1; min-width: 220px; background: #f8fafc; padding: 24px; border-right: 1px solid #e2e8f0;">
      <h4 style="margin: 0; font-size: 15px; font-weight: 600; color: #0f172a;">Net Cash Flow</h4>
    </div>
    <div style="flex: 2; min-width: 280px; padding: 24px;">
      <p style="margin: 0 0 12px 0; font-size: 15px; color: #334155; line-height: 1.5;">It is the actual difference between the money entering your bank account and the money leaving it in a specific month.</p>
      <p style="margin: 0; font-size: 14px; color: #64748b; line-height: 1.5;"><em>Importance:</em> A company can be profitable on paper but go bankrupt if its net flow is negative (because they sold everything on 90-day credit and today they can't make payroll).</p>
    </div>
  </div>

  <div style="display: flex; flex-wrap: wrap; border-bottom: 1px solid #e2e8f0;">
    <div style="flex: 1; min-width: 220px; background: #ffffff; padding: 24px; border-right: 1px solid #e2e8f0;">
      <h4 style="margin: 0; font-size: 15px; font-weight: 600; color: #0f172a;">Direct Labor</h4>
    </div>
    <div style="flex: 2; min-width: 280px; padding: 24px;">
      <p style="margin: 0 0 12px 0; font-size: 15px; color: #334155; line-height: 1.5;">The cost of personnel who are <strong>directly involved</strong> in producing the good or delivering the service. It is usually costed in "Man/Hours".</p>
      <p style="margin: 0; font-size: 14px; color: #64748b; line-height: 1.5;"><em>Example:</em> The salary of the developer writing the app code, or the factory worker assembling the part. The CEO's salary is <em>not</em> Direct Labor.</p>
    </div>
  </div>

  <div style="display: flex; flex-wrap: wrap; border-bottom: 1px solid #e2e8f0;">
    <div style="flex: 1; min-width: 220px; background: #f8fafc; padding: 24px; border-right: 1px solid #e2e8f0;">
      <h4 style="margin: 0; font-size: 15px; font-weight: 600; color: #0f172a;">Overhead</h4>
    </div>
    <div style="flex: 2; min-width: 280px; padding: 24px;">
      <p style="margin: 0 0 12px 0; font-size: 15px; color: #334155; line-height: 1.5;"><strong>Indirect Manufacturing or Operational Costs.</strong> These are the expenses necessary to keep the business alive, but which cannot be traced back to a single project.</p>
      <p style="margin: 0; font-size: 14px; color: #64748b; line-height: 1.5;"><em>Example:</em> Office rent, administrative salaries, electricity bills, and monthly subscriptions. They must be "prorated" in the budgets.</p>
    </div>
  </div>

  <div style="display: flex; flex-wrap: wrap; border-bottom: 1px solid #e2e8f0;">
    <div style="flex: 1; min-width: 220px; background: #ffffff; padding: 24px; border-right: 1px solid #e2e8f0;">
      <h4 style="margin: 0; font-size: 15px; font-weight: 600; color: #0f172a;">Budget vs. Actual (BvA)</h4>
    </div>
    <div style="flex: 2; min-width: 280px; padding: 24px;">
      <p style="margin: 0 0 12px 0; font-size: 15px; color: #334155; line-height: 1.5;">The ultimate financial control module. It compares the plan you drew up ("I thought I would sell $50,000 in January") against what actually happened by connecting to the bank account or Stripe transactions.</p>
      <p style="margin: 0; font-size: 14px; color: #64748b; line-height: 1.5; font-family: monospace; background: #f1f5f9; padding: 4px 8px; border-radius: 6px; display: inline-block;">Deviation Formula: (Actual - Budget) / Budget</p>
    </div>
  </div>

  <div style="display: flex; flex-wrap: wrap; border-bottom: 1px solid #e2e8f0;">
    <div style="flex: 1; min-width: 220px; background: #f8fafc; padding: 24px; border-right: 1px solid #e2e8f0;">
      <h4 style="margin: 0; font-size: 15px; font-weight: 600; color: #0f172a;">Starting / Ending Inventory</h4>
    </div>
    <div style="flex: 2; min-width: 280px; padding: 24px;">
      <p style="margin: 0 0 12px 0; font-size: 15px; color: #334155; line-height: 1.5;">Cord's production schedules are <strong>planning tools, not live inventory systems (ERP)</strong>.</p>
      <p style="margin: 0; font-size: 14px; color: #64748b; line-height: 1.5;">Therefore, the Starting Inventory is not read from a warehouse, but is an <em>assumption</em> you type in as a starting point so the engine calculates how many extra units you must manufacture to reach your goal.</p>
    </div>
  </div>

  <div style="display: flex; flex-wrap: wrap; border-bottom: 1px solid #e2e8f0;">
    <div style="flex: 1; min-width: 220px; background: #ffffff; padding: 24px; border-right: 1px solid #e2e8f0;">
      <h4 style="margin: 0; font-size: 15px; font-weight: 600; color: #0f172a;">NPV and IRR</h4>
    </div>
    <div style="flex: 2; min-width: 280px; padding: 24px;">
      <p style="margin: 0 0 12px 0; font-size: 15px; color: #334155; line-height: 1.5;"><strong>Net Present Value and Internal Rate of Return.</strong> Metrics used to evaluate if an investment is worthwhile. NPV discounts the time value of money; if it is greater than zero, the project is profitable.</p>
      <p style="margin: 0; font-size: 14px; color: #64748b; line-height: 1.5;"><em>Discount Rate:</em> The minimum yield % you demand from the project (usually the cost of your debt or the yield of a safe investment).</p>
    </div>
  </div>

  <div style="display: flex; flex-wrap: wrap; border-bottom: 1px solid #e2e8f0;">
    <div style="flex: 1; min-width: 220px; background: #f8fafc; padding: 24px; border-right: 1px solid #e2e8f0;">
      <h4 style="margin: 0; font-size: 15px; font-weight: 600; color: #0f172a;">Payback Period</h4>
    </div>
    <div style="flex: 2; min-width: 280px; padding: 24px;">
      <p style="margin: 0 0 12px 0; font-size: 15px; color: #334155; line-height: 1.5;">The exact time (in years and months) it will take for the project to generate enough cash flow to return your initial out-of-pocket investment.</p>
    </div>
  </div>

  <div style="display: flex; flex-wrap: wrap; border-bottom: 1px solid #e2e8f0;">
    <div style="flex: 1; min-width: 220px; background: #ffffff; padding: 24px; border-right: 1px solid #e2e8f0;">
      <h4 style="margin: 0; font-size: 15px; font-weight: 600; color: #0f172a;">Economic Order Quantity (EOQ)</h4>
    </div>
    <div style="flex: 2; min-width: 280px; padding: 24px;">
      <p style="margin: 0 0 12px 0; font-size: 15px; color: #334155; line-height: 1.5;">The perfect mathematical order size that minimizes total inventory costs (balancing what it costs to order freight vs. what it costs to store boxes).</p>
    </div>
  </div>

  <div style="display: flex; flex-wrap: wrap;">
    <div style="flex: 1; min-width: 220px; background: #f8fafc; padding: 24px; border-right: 1px solid #e2e8f0;">
      <h4 style="margin: 0; font-size: 15px; font-weight: 600; color: #0f172a;">Flexible Budget (Variances)</h4>
    </div>
    <div style="flex: 2; min-width: 280px; padding: 24px;">
      <p style="margin: 0 0 12px 0; font-size: 15px; color: #334155; line-height: 1.5;">A tool that analyzes why you spent more than planned, dividing the problem into two variances:</p>
      <ul style="margin: 0; padding-left: 20px; font-size: 14px; color: #64748b; line-height: 1.6;">
        <li><strong>Price Variance:</strong> You bought materials at a higher price than planned.</li>
        <li><strong>Efficiency Variance:</strong> You wasted more materials than planned to produce the same amount of units.</li>
      </ul>
    </div>
  </div>

</div>
