# Cord Elements (Componentes UI embebibles)

Cord Elements son componentes pre-construidos de React y Vanilla JS que te permiten incrustar la experiencia de Cord dentro de tu propio SAAS o página web sin tener que redirigir al usuario.

## Componentes Disponibles
- **`<PricingTable />`**: Un renderizador automático de planes y precios.
- **`<CheckoutElement />`**: Un formulario seguro que tokeniza la tarjeta del cliente bajo normativas PCI.
- **`<CustomerPortal />`**: Una vista incrustable para que el cliente gestione sus métodos de pago y descargue sus facturas.

Revisa la [Documentación Técnica de CORD Elements](/desarrolladores/elements) para ver ejemplos de implementación con Next.js y Vite.
