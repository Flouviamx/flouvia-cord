# Cumplimiento PCI-DSS

Cord opera como un Proveedor de Servicios de Nivel 1 bajo el estándar PCI-DSS (Payment Card Industry Data Security Standard).

## ¿Qué significa esto para ti?
No necesitas certificar tu empresa bajo PCI siempre y cuando uses nuestras herramientas (Enlaces de Cotización o Cord Elements). 

Nosotros tokenizamos la información confidencial. **Los datos crudos de las tarjetas nunca tocan tus servidores**, asegurando que estés completamente blindado ante posibles filtraciones.
