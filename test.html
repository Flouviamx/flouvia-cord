Preguntas frecuentes

¿Cord soporta flujos de aprobación interna para descuentos?

Sí. Puedes establecer umbrales lógicos. Si un representante ofrece un descuento mayor al permitido, la cotización se bloquea y requiere autorización de un gerente antes de poder ser enviada al cliente final.

¿Cómo se integra Cord con nuestro ERP actual?

Nuestra API REST y el sistema de Webhooks permiten sincronizar clientes, inventarios y listas de precios bidireccionalmente. También generamos un payload estándar para disparar la facturación en SAP, Oracle o el ERP de tu elección al aprobarse una cotización.

¿Podemos migrar nuestro catálogo de miles de SKUs?

Absolutamente. Cord está diseñado para escalar. Puedes importar vía CSV o conectar nuestra API para ingestar catálogos masivos. Las actualizaciones de precios se reflejan en tiempo real en todo el sistema sin afectar cotizaciones históricas.

¿Cuáles son los estándares de seguridad de la plataforma?

Todos los datos están encriptados en reposo y en tránsito. Las cotizaciones cerradas generan una firma hash SHA-256 que garantiza su inmutabilidad. Ofrecemos SLAs empresariales para disponibilidad y soporte técnico directo.
